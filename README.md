# Go Music JS

Go Music JS 是 `go-music-dl` 项目的 JS 插件版，面向 Songloft 插件系统使用。

本仓库已开源插件源码，并继续通过 GitHub Release 发布可直接安装的 `.jsplugin.zip` 插件包。

## 功能概览

- 多平台歌曲搜索：酷我、网易云、QQ、酷狗、咪咕、汽水等主要平台，并补充 Bilibili、JOOX、Apple Music、千千、5sing、Jamendo 等来源入口。
- 在线播放：支持搜索结果、推荐歌单、排行榜、我的平台歌单、本地歌单和本地音乐播放。
- 歌词与封面：统一多平台字段映射，尽量补全可播放歌曲的歌词、封面、歌手、专辑等元数据。
- Songloft 导入：支持将搜索、歌单、排行榜等来源的歌曲导入 Songloft，并尽量保留歌词、封面和来源信息。
- 本地能力：读取 Songloft 本地音乐和本地歌单，支持元数据、封面、歌词与播放。
- 推荐歌单：支持酷我、网易云、QQ、酷狗四个平台推荐歌单。
- 排行榜：支持酷我、网易云、QQ、酷狗四个平台排行榜。
- 我的歌单：根据已登录平台显示用户歌单，未登录时提示到设置中登录。
- 平台登录：支持 Cookie 登录，并支持网易云、QQ、酷狗、Bilibili 等平台扫码登录。
- 手动换源：歌曲无效或识别错误时，可打开换源弹窗自动查找相似歌曲，试听后确认替换。
- 日志窗口：设置页内置日志区域，便于查看 API、播放、歌词、导入、换源等运行状态。

## 使用说明

1. 在 [Release 页面](https://github.com/fanliu-yi/go-music-js/releases) 下载最新的 `go-music-js-v*.jsplugin.zip`。
2. 在 Songloft 插件管理中安装该 `.jsplugin.zip` 文件。
3. 打开 Go Music JS 插件页面后，可使用搜索、推荐歌单、排行榜、本地歌单、本地音乐和我的歌单。
4. 如需读取平台用户歌单，请进入插件设置页填写 Cookie 或使用支持的平台扫码登录。
5. 歌曲无法播放或识别错误时，点击歌曲旁边的“换源”按钮，试听候选歌曲后确认替换。

## 开发

```bash
npm install
npm run build
npm run validate
```

构建产物位于 `dist/go-music-js.jsplugin.zip` 或构建工具输出的对应插件包路径。
