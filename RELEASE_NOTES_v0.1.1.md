# Go Music JS v0.1.1 更新说明

对应历史开发版：`0.0.114`

## 主要更新

- 完成无容器内置音源版插件稳定化。
- 修复汽水音乐播放解密问题。
- 修复本地音乐读取、封面、歌词和播放体验。
- 修复本地歌单读取和播放。
- 完成网易云、QQ、酷狗、Bilibili 扫码登录基础支持。
- 调整设置页 UI，移除不必要的软件更新/容器连接配置。
- 增加日志窗口，便于查看播放、API、登录、导入等错误。
- 完成右上角操作提示，用于导入、保存设置等反馈。
- 优化底部播放栏 UI，保留音量控制。
- 优化推荐歌单和排行榜入口。
- 排行榜改为四平台强制显示：酷我、网易云、QQ、酷狗。
- 排行榜 UI 改为左侧榜单、右侧歌曲列表。
- 排行榜歌曲改为 30 首一页。
- 修复酷我排行榜歌曲无封面问题。
- 保留 Go 插件歌曲展示格式，不使用一字排开的表格样式。

## 已知边界

- `0.1.0` 对应历史开发版 `0.0.110`，已归档源码包和插件包。
- 下载功能暂未开发。
- 部分平台接口会随官方变动失效，需要后续维护音源适配逻辑。
