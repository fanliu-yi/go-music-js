# Go Music JS 功能说明

Go Music JS 是运行在 Songloft 中的内置音源音乐插件，不依赖独立 go-music 容器。

## 核心功能

- 多平台歌曲搜索：酷我、网易云、QQ、酷狗、咪咕、汽水等。
- 歌曲播放：支持平台歌曲、本地音乐、本地歌单歌曲播放。
- 歌词获取：播放时自动获取并显示歌词，支持本地音乐元数据歌词。
- 封面获取：搜索、推荐歌单、排行榜、导入歌曲尽量保留封面。
- 导入 Songloft：支持把歌曲导入 Songloft 曲库，并携带来源、封面、歌词等元数据。
- 本地音乐：读取 Songloft 本地音乐，支持封面、歌词和播放。
- 本地歌单：读取 Songloft 本地歌单并显示歌曲。
- 我的歌单：按已登录平台读取用户歌单。
- 推荐歌单：支持酷我、网易云、QQ、酷狗四个平台。
- 排行榜：支持酷我、网易云、QQ、酷狗四个平台，左侧榜单、右侧歌曲分页显示。
- 平台登录：支持 Cookie 登录；网易云、QQ、酷狗、Bilibili 支持扫码登录。
- 设置页面：支持下载目录、文件名模板、每页条数、元数据嵌入、平台 Cookie、日志查看。
- 日志窗口：支持 Info、Warn、Error、Trace 等日志筛选。

## 当前重点优化

- 酷我排行榜歌曲封面补全。
- 排行榜 30 首分页。
- 排行榜左右面板统一 UI。
- 保留 Go 插件歌曲展示格式。

## 使用边界

- 汽水音乐音频存在加密，插件内已做浏览器侧解密兼容。
- 部分平台接口字段不完整时，会使用搜索或专用接口补全封面、歌词等元数据。
- 下载功能当前保留入口，但点击提示“下载功能暂未开发”。
